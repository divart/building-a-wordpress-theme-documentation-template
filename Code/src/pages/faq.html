How do I change the footer copyright text?

To edit the footer, follow the next steps:

  1. Locate and open footer.php.
  2. Make the necessary changes and save the file.
  3. If you’re working on a copy of the file go ahead and upload it to your live server. Otherwise, if working locally, the changes will be visible after a page refresh.

Making changes to a .php file might cause problems. Always make sure you test first before deploying those changes to a live site. Also, make sure you always backup your WordPress site.
