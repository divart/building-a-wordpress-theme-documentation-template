What is covered by support?

As a buyer, it’s important to know what is and what isn’t covered by item support. Here’s a handy table that will give you all the information you need.

Description Coverage
Answering questions about how to use the item
Answering technical questions about the item (and included third party assets)
Help with defects in the item or included third party assets
Item updates to ensure ongoing compatibility and to resolve security vulnerabilities
Item customization
Installation of the item
Hosting, server environment, or software
Help from authors of included third party assets

For more information about support coverage, please visit the ThemeForest Item Support Policy.

How to get support

Support is only given via the support forums. Any request that comes through a different channel (email, social media, etc) will be ignored.

The only way to get support is via our dedicated support forums. The forums are the best way because they are searchable and chances are, whatever problem you’re having, someone else had it before you and it got fix. All you have to do is search for it and find the answer.

If you can’t find the answer go ahead and open a support ticket that will be followed up by one of our team members.
