Before you can use your brand new WordPress theme, you need to install it. For this, you need a working WordPress installation. If you don’t have one, please refer to the “Installing WordPress” page.

Avalanche requires WordPress version 4.5 in order to work properly. Installing it on an older version might result in unwanted behaviour.

{{> video-embed}} {{> nav-internal}}

Step 1. Locate the theme files

To install Avalanche, you must first locate the theme files. You can find these in the package you downloaded, in the zip file avalanche.zip (Fig. 1).

Step 2. Log into the WordPress back-end

A WordPress theme can only be installed from the administration area or back-end. To access it, you must login by accessing the main address of your website and adding /wp-login.php. For example, if your website’s address is http://www.mysite.com, then your can access the administration menu by first loggin in at this address: http://www.mysite.com/wp-login.php.

If you're having difficulties accessing the login page, please contact the party responsible for installing WordPress on your domain.

Once you’ve accessed the interface, simply fill in your username and password (Fig. 2).

Step 3. Install Avalanche

The zip file must NOT be extracted in order to install the theme. It is uploaded as a single file.

To install the theme, follow these steps in order:

  1. Navigate to Appearance > Themes.
  2. Click the Add New button.
  3. Click the Upload Theme button.
  4. Click the Browse... button and select the avalanche.zip file you located earlier.
  5. Once installed, open the Themes page again, select the newly installed theme and click on the Activate button.
  6. Enjoy your new WordPress theme!
